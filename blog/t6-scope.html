Terminology: "Learning the Lingo"
Scope

Scope is the context in which values and expressions are "visible," or can be referenced to in your code.

If a variable or other expression is not "in the current scope," then it is unavailable for use.

In Javascript there is two types of scope: local and global.

Variables and expressions defined in the Global scope can be accessed anywhere in your code. Global scope exists before you write any code.

Local scope is the scope within functions or {}, it can only be accessed within that function. If you try and access it outside that function, you console will return "undefined" as it is not defined outside that function (i.e. in the Global scope.)

When you have functions within functions. For example outerFunction and innerFunction. The outerFunction will not be able to access the variables defined in innerFunction as var in innerFunction are local to innerFunction but innerFunction will be able to access variables in the outerFunction as it is considered global to innerFunction.
