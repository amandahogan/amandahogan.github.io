Terminology: "Learning the Lingo"

What is grid based design?

Our Eyes naturally follow a pattern from left to right then down. A grid based layout is columns and rows in a framework

Whats all the hype about responsive webpages?

Responsive webpages are pages that are designed to change the width of the columns and webpage to shrink or expand depending on what screen you are viewing the platform on. Its a way to adjust websites so that it can be viewed efficiently on all types of screens and devices i.e. your mobile phone, tablet, computer or TV.

What is semantic structure?

Semantic structure is using tags that is not only understood by your OS but by humans too. More simply put its a structure with meaning.
